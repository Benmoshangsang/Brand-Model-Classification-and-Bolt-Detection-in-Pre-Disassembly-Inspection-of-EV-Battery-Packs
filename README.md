# Brand-Model-Classification-and-Bolt-Detection-in-Pre-Disassembly-Inspection-of-EV-Battery-Packs
This repository contains code for lithium battery brand/model classification and bolt detection using the Mamba-CSDS framework. It includes training, evaluation, ablation studies, and robustness testing under varying lighting and motion blur. The code is compatible with Python 3.10 and PyTorch 2.1.2.
